Los modelos de esta sección están bajo desarrollo para su futura implementación en el modelo principal del sistema solar o en otros modelos secundarios. Asimismo se pretende que sirvan como práctica para aquellos que deseen aprender a usar three.js y WebGl

Asteroide Bennu

Modelo y animación

Este projecto ejecuta el cargador para objetos .stl usado frecuentemente en impresión 3d. También analiza el rendimiento y la apariencia de objetos con una iluminación falsa.

Comparación del tamaño de los planetas

Modelo y animación

Este proyecto muestra los planetas del sistema solar de manera que se pueda apreciar la diferencia de tamaño entre ellos. El ambiente hace uso de una iluminación falsa.

Elementos orbitales

Modelo interactivo

Este proyecto muestra un diagrama interactivo de los elementos orbitales para una orbita elíptica. Estos elementos se integrarán en cada objeto del modelo principal y su representación será controlada mediante el lenguaje Astra.

Bóveda Celeste

Modelo interactivo

Este proyecto muestra la bóbeda celeste tal como es generada para el modelo principal AstraSolaris. Su finalidad es la de desarrollar los marcadores y líneas de referencia adecuadas para su integración en el modelo principal.

Tabla Periódica

Modelo interactivo

Este projecto presenta una talbla periódica en diferentes disposiciones de los elementos. Servirá como base para una aplicación relacionada con espectroscopía astronómica. También tiene como objetivo desarrollar el código para la manipulación de etiqueteas 3d en el modelo principal de Astra Solaris.
